Aprendizagem sobre as TAGS

(Voltar)

O primeiro ponto que foi abordado na minha aprendizagem ao HTML, foi a questão das tags. A estrutura do site depende muito de um conhecimento sólido em relação a esse assunto. Os iniciantes, com toda certeza, terão dificuldades para aprender a função de cada Tag e sua descrição, porém, garanto que é a base para fazer um bom trabalho quando se trata de HTML.

Para utilizar as tags é importante obdecer o formato correto para que se execute a função pretendida. Podemos citar alguns exemplos de tags: head - seria a cabeça do seu trabalho, ou seja, a descrição inicial do seu projeto; title - essa função trás um título do seu projeto; body - seria o desenvolvimento do seu trabalho; strong - essa função inclui o negrito no texto; i - inclui a função itálico no projeto; input - entrada de alguma informação de tipos variados; img - inclusão de imagens no seu projeto;

É importante salientar que ao usar uma tag, temos que abrir e fechar a mesma: < + tag + />

Elaboração de textos

(Voltar)

A produção dos textos foi um dos principais pontos aprendidos no curso de Introdução ao HTML, pois tem uma grande utilização quando se trata de sites, blogs e artigos de opinião. A construção de um bom texto depende de um aprendizado sólido sobre as tags.

Quando construimos um texto podemos colocar diversos elementos absorvidos no curso de Introdução ao HTML. A ideia da utilização desses elementos é deixar o conteudo a ser lido, cada vez mais, interessante ao leitor ou usuário da informação.

Produção de listas ordenadas e não ordenadas

(Voltar)

A utilização de listas ordenadas e não ordenadas se tornam muito importânte para a elaboração de índices. Isso serve para que o usuário da informação tenha uma experiência agradável ao explorar o conteúdo abordado naquela página acessada.

Os formatos para as Listas Ordenadas são o < +ol + lista + /> e para as Listas Não Ordenadas se utiliza < +ul + lista + />.

Criação de Links

(Voltar)

A introdução dos links na estrutura da linguagem HTML pode tornar o trabalho de um Developer HTML mais robusto. Uma boa utilização deles pode influênciar na compreensão das informações presentes naquele site. Além de que, através dos links pode ser incluídos assuntos presentes na internet para um trabalho que está sendo elaborado.

Para a utilização dos links deve ser utilizado a tag: a + href= + a. Com isso, a pessoa que está criando a estrura pode colocar dentro dessa tag o conteúdo que irá sair no formato de link. OBS: Sempre utilizando a estrura das tags (< + />)
